Для создания программ, которые сами могут выполнить больше действий чем программист, используются циклы. Циклы позволяют множество раз выполнить определенные команды.

Для создания цикла с условием в языке Python 3 используется конструкция while

while условие:

действия

действия в теле цикла будут повторяться до тех пор пока условие цикла истинно. Как только действия завершились, условие будет проверяться вновь. Потенциально действия цикла могут выполняться бесконечно, если условие всегда истинно:

while True:

действия в вечном цикле

Также действия цикла while могут не выполниться ни разу, если условие сразу равно False. Конструкция while похожа на конструкцию if. В некотором смысле можно сказать, что while -- это бесконечно повторенный if. Однако к while нельзя добавлять блоки else и elif, так как это не имеет смысла.

Например, с помощью цикла while можно перебрать все числа от 1 до 10:

n = 1

while n <= 10:

print(n)

n = n + 1

C помощью циклов можно организовать вывод длинных последовательностей на экран и чтение с экрана длинных последовательностей. Так для того, чтобы считать с экрана последовательность чисел, можно считывать и обрабатывать элементы этой последовательности один за одним в цикле:

while True:

x = int(input()) #считать очередное число x

# обработать число x

Также существуют две команды для манипуляции с циклами: break -- прерывает текущий цикл и переходит сразу к инструкциям после цикла, continue -- завершает текущую итерацию цикла и переходит к проверке условия для следующей итерации. Таким образом с помощью break, можно завершить вечный цикл с условием True, который в ином случае не завершится никогда.

Вы можете вкладывать в действия цикла любые другие конструкции в любом количестве -- внутрь цикла можно вложить условия и другие циклы. Цикл, внутри цикла называется внутренним. Цикл, содержащий другой цикл в себе, называется внешним. Для вложенных циклов операции break и continue работают для самого внутреннего цикла.

В стандартном режиме, каждая команда print, выводит данные на новой строке. Однако циклы можно использовать для вывода сложных конструкций на одной строке, в таком случае можно указать параметр end для print, если указать end='', то перевода на новую строку не будет и можно с помощью нескольких команд print выводить информацию в одну строку.

print(1, end='')
